# Getting Started

## Requirements

- Node.js 20+
- Bun 1.3+

## Install

```bash
bun install
```

## Common Commands

```bash
bun run build
bun run typecheck
bun run test
bun run pack:package
```

## Imports

Install the published package:

```bash
npm install @futelab/vop
```

Main imports:

- `@futelab/vop/sdk`
- `@futelab/vop/protocol`
- `@futelab/vop/runtime`
- `@futelab/vop/panel/react`

## Runtime File

```bash
bunx @futelab/vop generate --config ./vop.config.ts
```
